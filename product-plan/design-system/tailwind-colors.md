# Tailwind Color Configuration

## Color Choices

- **Primary: emerald** — Active nav, brand accents, positive signals
- **Secondary: amber** — Warnings, under-patrolled highlights, secondary badges
- **Neutral: stone** — Backgrounds, borders, body text (warm gray)

## Usage Examples

- Primary emphasis: `bg-emerald-600 text-white`, `text-emerald-600 dark:text-emerald-400`
- Secondary / caution: `bg-amber-50 text-amber-800`, `border-amber-200`
- Neutral chrome: `bg-stone-50 dark:bg-stone-950`, `text-stone-600 dark:text-stone-400`, `border-stone-200 dark:border-stone-800`
